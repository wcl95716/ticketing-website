# API 文档

## 接口

### `/api_endpoint`

描述：这是来自 api_module.py 的一个API端点。

请求：
- 方法：GET
- 端点：`/api_endpoint`

响应：
- 成功：字符串 "This is an API endpoint from api_module.py"

### `/add_chat_record` (POST)

描述：添加聊天记录。

请求：
- 方法：POST
- 端点：`/add_chat_record`
- 请求体：包含聊天记录数据的 JSON 对象

响应：
- 成功：JSON 响应，包含消息 "Chat record added successfully"
- 错误：JSON 响应，包含错误消息

### `/get_chat_history/<ticket_id>` (GET)

描述：获取特定票证的聊天历史记录。

请求：
- 方法：GET
- 端点：`/get_chat_history/<ticket_id>`
- URL 参数：`ticket_id` - 要检索聊天历史记录的票证ID

响应：
- 成功：JSON 响应，包含聊天历史数据
- 错误：JSON 响应，包含错误消息

### `/add_ticket` (POST)

描述：添加票证。

请求：
- 方法：POST
- 端点：`/add_ticket`
- 请求体：包含票证数据的 JSON 对象

响应：
- 成功：JSON 响应，包含消息 "Ticket added successfully"
- 错误：JSON 响应，包含错误消息

### `/get_ticket/<ticket_id>` (GET)

描述：获取特定票证的详细信息。

请求：
- 方法：GET
- 端点：`/get_ticket/<ticket_id>`
- URL 参数：`ticket_id` - 要检索票证的ID

响应：
- 成功：JSON 响应，包含票证详细信息
- 错误：JSON 响应，包含错误消息

### `/get_all_tickets` (GET)

描述：获取所有票证的列表。

请求：
- 方法：GET
- 端点：`/get_all_tickets`

响应：
- 成功：JSON 响应，包含所有票证的列表数据
- 错误：JSON 响应，包含错误消息
